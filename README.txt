Algumas convenções utilizadas ( pelo Ivandro )

formato dos comentários do cabeçalho das funções

/*
** name: // nome da função
** description: // pequena descrição da função
** detailed description: // descrição detalhad da função, descreve em promenor todas as responsabilidades da função. nota: opcional
** parameters: // pârametros da função. nome e pequena descrição. nota: é omitido se função não tiver pârametros 
** return: // valor de retorno. pequena descrição. nota: é omitido se função retornar void
*/

formato das váriaveis

váriavel de uma classe: m_nomeDaVariavel // m de member
váriavel global: g_nomeDaVariavel // g de global
váriavel statica: s_nomeDaVariavel // s de static
